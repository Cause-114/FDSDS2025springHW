# DCT 图像压缩模拟项目

本项目为基于 DCT（离散余弦变换）的图像压缩模拟，使用 MATLAB 实现。模拟流程涵盖
从图像采样、颜色空间转换、DCT 变换、量化、霍夫曼编码，到压缩效果展示与性能评价。

---

## 当前功能

- 支持压缩图像效果展示；
- 提供 PSNR、SSIM 等压缩质量评价指标；
- 支持理论 Bit 数计算（经采样-DCT-量化-霍夫曼编码流程后纯数据大小）；

---

## 文件结构说明

- `Code/`：主代码文件夹
  - `main.m`：主程序入口，运行此文件即可开始图像压缩模拟。
  - 其余 5 个.m文件为支撑模块（定义了一些类和功能函数），请确保与 `main.m` 位于同一目录下。
  - `JustFunCode/`: 里面有几个我写实验报告时顺手写的几个小代码（包含了实验报告中重点提过的`DCT_Realize.m`）
  ，独立于主流程。每个.m文件独立运行，均附注释说明功能，感兴趣可以单独尝试。
  - `TestFigure/`: 测试用图，具体请见`main.m`的注释。

- `report/`：实验报告文件夹
  - 包含完整的实验报告 PDF、LaTeX 源码和所有插图。

---

## 使用说明

1. **运行程序**：  
   打开 MATLAB，进入 `Code` 文件夹，运行 `main.m` 即可模拟图像压缩流程。

2. **定制运行内容**：  
   `main.m` 中可根据自身需要选择执行不同模块（如仅进行压缩或仅计算指标等）。
