## 数值算法与案例分析 I、II作业代码+PJ
数值一主要是讲数值线性代数，数值二主要是数值逼近。
### 作业代码
数值一的题目已经找不到了。每次作业的证明题等是手写的，所以没有放上去。代码读起来可能没那么优雅，但是确实能运行。

主要内容：
- 数值一：LU、QR分解；特征值，对称特征值问题；稀疏线性系统求解。
- 数值二：非线性方程数值解法；插值拟合；数值积分微分；快速傅里叶变换；微分方程数值解法。
### PJ

- PJ3: 数值算法2的PJ，实现了基于DCT的图像压缩（事实上没有将其保存成为格式，只是理论上推导了目标文件大小，烂尾了）。
